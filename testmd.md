중앙 처리 장치 : CPU (central processing unit) 

산술 논리 장치 : ALU (arithmetic login unit)

입력

1. 피연산자
2. 명령코드
    
    clr, set, not, neg, shl, shr, load, and, or, xor, add, sub, cmp
    

출력

1. 결과
2. 조건코드
